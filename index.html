EcoZone v2.4.1 · Parque Linear Criciúma
EcoZone
Protótipo de app mobile para gestão sustentável de área verde com captação de chuva e monitoramento de carbono.

Como navegar

Clique nos botões da barra inferior para trocar de tela. Os cards do Dashboard também são clicáveis.

Telas disponíveis

Dashboard — Score ambiental, resumo de todos os indicadores e alertas ativos

Água — Nível do reservatório animado, histórico e previsão de chuva

Irrigação — Controle por zonas, modo automático e sugestão da IA

Carbono — Captura de CO₂, gauge animado e conversão em energia equivalente

Perfil — Estatísticas do gestor e configurações

Stack sugerido para o app real

React Native + Expo · Node.js + MQTT para IoT · InfluxDB para séries temporais · API OpenWeatherMap · TensorFlow Lite para sugestões de irrigação
